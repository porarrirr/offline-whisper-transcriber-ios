# Whisper文字起こし

**iPhone だけで動く、プライバシー重視の AI 音声文字起こし**

OpenAI Whisper を端末内で実行。録音もファイルインポートも、モデル取得後はオフラインで文字起こしできます。

> **ご注意:** 文字起こし結果の正確性は保証されません。重要な判断の唯一の根拠にせず、必ず内容をご確認ください。[免責事項](disclaimer.html)

## 主な機能

| 機能 | 説明 |
|------|------|
| リアルタイム録音 | マイクで録音し、そのまま文字起こし |
| ファイル対応 | m4a / wav / mp3 / mp4 / mov などをインポート |
| 多言語 | 日本語を含む約 20 言語 |
| オフライン推論 | モデルダウンロード後はインターネット不要 |
| 履歴管理 | 検索・お気に入り・エクスポート |
| プライバシー | 音声・結果を開発者サーバーへ送信しない |

## こんな方におすすめ

- 会議やインタビューのメモを端末内だけで残したい
- クラウドに音声を上げずに文字起こししたい
- オフライン環境でも議事録を作りたい

## 動作環境

- **iOS 17.0 以降**（実機のみ。Simulator 非対応）
- 初回起動時に Whisper モデル（約 142MB）のダウンロードが必要
- マイク利用時はマイクへのアクセス許可が必要
