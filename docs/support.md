# Whisper文字起こし — サポート

オフライン対応の AI 音声文字起こしアプリ（iOS）のサポートページです。

[マーケティングページに戻る](index.html)

## 動作環境

- iOS 17.0 以降（実機のみ。Simulator 非対応）
- インターネット: 初回の Whisper モデルダウンロード時のみ必要。ダウンロード完了後はオフラインで文字起こし可能

## 初回セットアップ

1. アプリを起動するとモデルダウンロード画面が表示されます
2. デフォルトの Base モデル（約 142MB）を Hugging Face からダウンロードします
3. 完了後、録音またはファイルインポートで文字起こしを開始できます

VAD（無音スキップ）を使う場合は、設定から別途 VAD モデルのダウンロードが必要です。

## よくある質問

### モデルのダウンロードに失敗する

Wi-Fi 接続を確認し、ストレージ容量に余裕があるか確認してください。設定画面から再ダウンロードできます。

### マイクが使えない

設定アプリでマイクの許可をオンにしてください。初回録音時に許可ダイアログが表示されます。

### 文字起こしが遅い / メモリ不足になる

設定でより小さいモデル（Tiny など）を選ぶか、Flash Attention を有効にしてください。長時間の音声は端末のメモリによって失敗することがあります。
